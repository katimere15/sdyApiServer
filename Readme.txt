프로젝트 환경 설명서

## 프로젝트 개요
- 본 프로젝트는 Spring Boot 기반의 백엔드 API 서버 애플리케이션입니다.
- 폐쇄망 환경(인터넷 연결 없음)에서 동작하는 것을 가정합니다.

## 주요 폴더 구조
- src/main/java : 자바 소스 코드
- src/main/resources : 설정 파일 및 리소스
- libs : 외부 라이브러리(JAR) 보관
- build : 빌드 결과물

## 환경 특징
- 외부 인터넷 연결이 없으며 내부 네트워크 내에서만 통신 가능
- 외부 API 호출 및 라이브러리 다운로드 제한
- 필요한 외부 라이브러리는 미리 준비하여 libs 폴더에 보관

## 개발 및 운영 기본 지침
- 개발과 테스트는 폐쇄망 환경을 가정하여 진행
- 배포 시에도 동일한 환경을 유지할 것
